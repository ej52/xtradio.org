# xtradio.org
XTRadio.ORG - Main website


## Local development/testing

### Install dependencies

```bash
npm i
```

### Run dev server


```bash
npm start
```

### Build application

```bash
npm run build
```

## Production build

### Build docker image

```bash
sh ./build.sh
```

### Build and run docker image

```bash
docker-compose up -d
```
